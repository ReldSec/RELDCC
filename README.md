Este es un script sencillo para generar tarjetas de credito a partir de otra o de un bin.
El uso esta bajo su responsabilidad, el proyecto puede ser compilado en cualquier sistema operativo.
En el caso de Linux abra una terminal y escriba;

gcc reldcc.c -o relcc


./reldcc
